## Release Schedule

Obsidian Chess Studio follows a continuous release model:

- **Patch releases** (x.x.X): Bug fixes and minor improvements, released as needed
- **Minor releases** (x.X.0): New features and enhancements, released monthly
- **Major releases** (X.0.0): Significant changes and major features, released quarterly

## How to Update

<Steps>
  <Step title="Check for updates">
    The app will notify you when a new version is available, or you can manually check for updates in **Settings → About**.
  </Step>
  <Step title="Download the update">
    Click the update notification or visit the [releases page](https://github.com/luisrivasnoriega/Obsidian-Chess-Studio/releases/latest).
  </Step>
  <Step title="Install the new version">
    - **Windows/Linux:** Download and run the installer
    - **macOS:** Download the .dmg and replace the app
    - **Android APK:** Download the .apk and install (requires "Install unknown apps" permission)
  </Step>
  <Step title="Restart the application">
    Your settings, games, and databases will be preserved.
  </Step>
</Steps>

<Note>
  Your data is stored separately from the application, so updating won't affect your games, profiles, or settings.
</Note>

## Android Updates

If you installed Obsidian Chess Studio on Android via direct APK (not from an app store):

1. Download the latest `.apk` from the [releases page](https://github.com/luisrivasnoriega/Obsidian-Chess-Studio/releases/latest)
2. Open the file to launch the system installer
3. Grant "Install unknown apps" permission if prompted
4. Install the update

## Breaking Changes

We strive to maintain backward compatibility. When breaking changes are necessary, they will be:

- Clearly documented in release notes
- Announced in advance when possible
- Accompanied by migration guides

<Warning>
  Major version updates (e.g., 2.x to 3.x) may include breaking changes. Always read the release notes before upgrading.
</Warning>
